# Brigata for Android

Official Android release feed for **Brigata**. The app source lives in a private repo; only built APKs are published here.

## Install with Obtainium (recommended)

1. Install [Obtainium](https://github.com/ImranR98/Obtainium) on your Android phone.
2. Add App -> paste this URL: `https://github.com/0xC/brigata-mobile`
3. Obtainium installs the latest APK and auto-updates when new versions ship.

## Or sideload directly

Download the `.apk` from the latest [release](https://github.com/0xC/brigata-mobile/releases) and open it on your phone (allow install from unknown sources).

> Early builds. Sign-in and chat arrive in upcoming versions.
